Interogarea 6.b.

Cerinta colocviu partial: Să se găsească pentru fiecare bătălie numărul claselor din fiecare categorie rezultat.

Cod SQL:
SELECT c.batalie, c.rezultat, COUNT(n.clasa) AS "Numarul claselor"
FROM nave n JOIN consecinte c ON (n.nume = c.nava)
GROUP BY batalie, rezultat
ORDER BY batalie;
